Employee Attrition Prediction System

This repository contains a Flask + MySQL app that serves a prediction form and stores results. GitHub Pages can only host static files, so this page is a static landing page for the project.

Run locally (recommended)

  1. Set up MySQL and create the schema from database_setup.sql.
  2. Create env vars for DB credentials (see .env.example).
  3. Install Python deps and start the Flask server: python app.py.
  4. Open http://127.0.0.1:5000.

What’s on GitHub Pages?

This landing page (no backend). To host the full app publicly, deploy the Flask server somewhere like Render/Railway/Fly.io and point the frontend to that API.

Tip: you can still browse the source code in the repo, including app.py, templates/, and static/.
